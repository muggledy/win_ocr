1. install python from https://www.python.org/downloads/ and pip install baidu-aip
2. set your APP_ID, API_KEY and SECRET_KEY in baidu_ocr.py
3. install AutoHotKey from https://www.autohotkey.com/ and double click to execute myhotkey.ahk
4. now you can take screenshots interactively via shortcut keys Ctrl+Alt+X and trigger recognizing text by closing SnippingTool, the recognition results are written to the clipboard
