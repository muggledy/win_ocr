1. Install python from https://www.python.org/downloads/ and pip install baidu-aip
2. Set your APP_ID, API_KEY and SECRET_KEY in baidu_ocr.py
3. Install AutoHotKey from https://www.autohotkey.com/ and double click myhotkey.ahk
4. Now you can take screenshots interactively in 15 seconds via shortcut keys Ctrl+Alt+X, the recognition results will be written into the clipboard. If there is no screenshot operation within 15 seconds, the text recognition program will terminate. You need to press the shortcut keys again to take a new screenshot for text recognition
